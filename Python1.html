Синтаксис Python

Python выделяется простотой синтаксиса. Основные особенности:

  • Отступы вместо скобок (4 пробела для блоков)
  • Динамическая типизация (переменные не требуют объявления типа)
  • Простые конструкции: функции через def, циклы for с коллекциями

Сравнение с другими языками

Особенность Python JavaScript C++/C# Java
Отступы/скобки Отступы (4 пробела) {} {} {}
Типизация Динамическая Слабая Статическая Статическая
Объявление переменных x = 5 let x = 5 int x = 5 int x = 5

Ключевые отличия

Python vs JavaScript

Python короче и читабельнее, JS гибче для сложных конструкций

Python vs C++/C#

Python медленнее из-за динамической типизации, но автоматически управляет памятью

Python vs Java

Java требует больше кода (например, public class), Python проще для начинающих

Примеры кода

Python


def greet(name):
    print(f"Привет, {name}!")

JavaScript


function greet(name) {
    console.log(`Привет, ${name}!`);
}

C++


#include <iostream>
using namespace std;

void greet(string name) {
    cout << "Привет, " << name << "!";
}

Java


public class Greeter {
    public static void greet(String name) {
        System.out.println("Привет, " + name + "!");
    }
}
