Основные особенности Swift

Swift — современный язык от Apple для разработки приложений под iOS, macOS, watchOS и tvOS. Основные его особенности:

  • Мультипарадигменность — поддержка императивного, функционального и объектно-ориентированного программирования.
  • Статическая типизация — строгая проверка типов на этапе компиляции.
  • Безопасность — защита от `NullPointerException` через опциональные типы и операторы `?.`, `?:`.
  • Автоматическое управление памятью — система ARC (Automatic Reference Counting).
  • Высокая производительность — компиляция в машинный код, оптимизации для мобильных устройств.
  • Упрощённый синтаксис — отсутствие точек с запятой, сокращённые конструкции (например, `let`/`var`).

Ключевые отличия

Swift vs Objective-C

Swift устраняет избыточность Objective-C: сокращает код на 40% за счёт умных преобразований типов и сокращённых конструкций (например, `data class`).

Swift vs Python

Swift статически типизирован и безопаснее, Python — динамичен, но подвержен ошибкам с `null`.

Swift vs JavaScript

Swift ориентирован на мобильные приложения, JavaScript — на веб. Swift требует компиляции, JavaScript — интерпретируется.

Swift vs Kotlin

Swift совместим с Objective-C, Kotlin — с Java. Оба поддерживают функциональные паттерны, но Swift акцентирует безопасность.

def greet(name):
    print(f"Привет, {name}!")

JavaScript


function greet(name) {
    console.log(`Привет, ${name}!`);
}

Kotlin


fun greet(name: String) {
    println("Привет, $name!")
}
